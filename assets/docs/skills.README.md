# 用户自定义技能目录

此目录用于存放用户自定义的 Skill。

你可以将包含 `SKILL.md` 的技能文件夹放置于此。

重新启动系统后，系统将自动加载这些技能并供 Agent 选用。

---

# User Skills Directory

This directory is used to store user-defined Skills.

You can place your skill folders containing `SKILL.md` here.

After restarting the system, these skills will be automatically loaded and available for the Agent to use.

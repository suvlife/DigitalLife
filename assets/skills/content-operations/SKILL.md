# 内容运营与发布

## 适用范围
- 内容选题、排期、多平台分发、发布执行与效果复盘、活动/社区运营。

## 工作流程
1. 依据目标（拉新/活跃/转化）制定选题与内容日历。
2. 针对各平台适配标题、封面、正文与话题标签。
3. 规划发布时间与频次，准备互动与分发策略。
4. 发布后回收数据（曝光/互动/转化），对比目标复盘。
5. 沉淀爆款要素与可复用模板，迭代下一轮选题。

## 产物与规范
- 交付内容日历 + 分发方案 + 复盘报告（可用 spreadsheet-studio 落表）。
- 文案创作交 copywriting，正式文档交 document-studio。

## 依赖与降级
可结合平台数据；无数据接口时以人工记录的关键指标复盘。

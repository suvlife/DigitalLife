# 项目与交付管理

## 适用范围
- 制定计划、拆解与分派任务、跟踪进度与风险、组织复盘。

## 工作流程
1. 明确目标、范围、交付物与验收标准（先对齐再排期）。
2. 用 WBS 拆解任务，标注负责人、工期、依赖与里程碑。
3. 识别风险与阻塞，给出应对与责任人，定期更新状态。
4. 用简明看板/清单跟踪：待办/进行/阻塞/完成。
5. 阶段结束做复盘：目标达成、偏差原因、改进项。

## 产物与规范
- 交付项目计划 + 里程碑表 + 风险清单（可用 spreadsheet-studio 落表）。
- 团队内可配合任务协作工具分派与跟踪。

## 依赖与降级
以清单/表格交付；无专用 PM 工具时用结构化文本管理。

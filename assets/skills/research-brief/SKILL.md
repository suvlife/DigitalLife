# 研究与信息检索

## 适用范围
- 就某一主题做背景调研、事实核查、竞品/政策/行业资料检索。
- 将零散资料综合为可引用、可复核的研究简报。

## 工作流程
1. 先把问题拆成 3-6 个可检索的子问题，明确时间范围与地域限定。
2. 使用 `web_search` / `web_fetch` 及可用数据检索工具，一次只查一个明确意图、一个主体。
3. 对关键结论至少交叉验证两个独立来源，记录每条事实的出处（标题 + URL + 日期）。
4. 区分「事实」「推断」「观点」，对不确定信息显式标注置信度。
5. 输出研究简报：结论摘要 → 关键发现（带引用）→ 分歧与空白 → 下一步建议。

## 产物与规范
- 简报开头给出 3-5 条结论要点；正文每条论据附来源。
- 引用统一列在文末「参考来源」，编号可回溯。
- 需要正式文档/表格时，转交 document-studio / spreadsheet-studio 落地。

## 依赖与降级
可用联网检索工具即可；无联网时应明确说明信息基于既有知识、可能过时。

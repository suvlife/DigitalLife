# UI/UX 设计

## 适用范围
- 输出交互流程、界面布局说明、设计规范与组件定义。

## 工作流程
1. 从用户目标与关键任务出发，梳理信息架构与主流程。
2. 设计页面结构与状态（空/加载/错误/成功），标注交互反馈。
3. 定义视觉规范：栅格、间距、字体层级、色彩、组件规则。
4. 兼顾可用性与可访问性（对比度、可点击区域、反馈时延）。
5. 输出可交付给前端的设计说明与验收要点。

## 产物与规范
- 交付交互说明 + 布局描述 + 设计规范；需网页原型时可配合 frontend-design。

## 依赖与降级
以文字化设计规范交付即可；不依赖专有设计工具。

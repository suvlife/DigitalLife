# 表格制作技能

## 适用范围
- 读取 `.xls`、`.xlsx`、`.csv` 数据。
- 生成可下载的 `.xlsx` 工作簿。
- 制作测算表、任务清单、数据分析、排期和汇总图表。

## 工作流程
1. 从团队工作目录 `uploads/` 读取上传表格，先确认工作表名称、行列数、字段类型和缺失值。
2. `.xlsx` 使用 `openpyxl`；`.xls` 优先用 LibreOffice 转换或 `xlrd` 读取；CSV 明确编码。
3. 新建工作簿时必须设置：醒目的表头、合理列宽、冻结首行、自动筛选、数字格式和打印区域。
4. 公式应写入单元格而不是把计算结果写死；重要汇总增加校验公式。
5. 数据适合时创建柱状图、折线图或饼图，避免无意义装饰。
6. 输出到 `outputs/`，重新加载工作簿检查工作表、公式及关键单元格。
7. 回复中给出相对下载路径。

## 依赖
使用 `openpyxl`；需要数据分析时可使用 `pandas`。依赖不可用时输出 CSV 与 Markdown 数据说明作为降级方案。

# 财务与投资分析

## 适用范围
- 财报分析、公司估值、投资逻辑构建、风险与投后跟踪。

## 工作流程
1. 先核对财务数据来源与口径（会计准则、期间、合并范围）。
2. 分析三张表与关键比率：盈利、成长、质量、偿债、现金流。
3. 估值用 DCF / 可比公司 / 可比交易，写清假设与敏感性。
4. 构建投资逻辑：驱动因素、护城河、催化剂、反面情形。
5. 明确风险与不确定性，给出置信度而非绝对判断。

## 产物与规范
- 交付分析结论 + 关键数据 + 估值区间 + 风险清单。
- 数据模型交 spreadsheet-studio，报告交 document-studio。

## 依赖与降级
不提供个性化投资建议式承诺；结论均为分析性判断并标注假设。

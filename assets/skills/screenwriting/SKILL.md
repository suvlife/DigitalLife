# 编剧与剧本创作

## 适用范围
- 构思故事、搭建结构、写人物与对白、产出大纲与剧本。

## 工作流程
1. 先定 logline（一句话故事）与主题、目标受众与时长。
2. 搭三幕结构：建置—对抗—解决，标注关键转折点。
3. 设计人物：目标、动机、障碍、弧光；关系推动冲突。
4. 分场写作：场景标题 + 动作描述 + 对白，保持节奏与信息密度。
5. 自检：冲突是否升级、动机是否可信、主题是否贯穿。

## 产物与规范
- 交付 logline → 大纲 → 分场剧本；分镜落地交 storyboard-design。

## 依赖与降级
纯创作，无外部依赖；改编真实题材时先经 research-brief 核实事实。

# 政策与公文起草

## 适用范围
- 设计政策方案、合作框架，撰写政府汇报材料与规范公文。

## 工作流程
1. 明确政策目标、适用对象、上位依据与边界，避免与现行法规冲突。
2. 设计条款：目标 → 适用范围 → 支持措施 → 申报/兑现流程 → 监管。
3. 汇报材料按「背景—问题—方案—收益—落地」组织，数据可核。
4. 使用规范、克制、可执行的公文语言，避免夸大与歧义。
5. 涉及资金/优惠须提示财务可持续性与合规风险。

## 产物与规范
- 交付政策要点/条款草案 + 汇报材料；正式排版交 document-studio。

## 依赖与降级
政策依据先经 research-brief 核实；合规风险交 compliance-review 复核。

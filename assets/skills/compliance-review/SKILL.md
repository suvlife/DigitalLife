# 合规风控与审校

## 适用范围
- 对方案/内容/交付物做合规、风险与文字审校。

## 工作流程
1. 明确适用规则（法规、平台规则、品牌红线）与审查范围。
2. 合规审查：资质、承诺表述、数据与隐私、知识产权、广告法禁用词。
3. 风险识别：财务可持续性、声誉、法律、操作风险，做分级。
4. 文字审校：事实一致、逻辑通顺、口径统一、错别字与格式。
5. 输出问题清单 + 风险等级 + 整改建议，最严重项排前。

## 产物与规范
- 交付审查意见：问题 → 依据 → 风险级别 → 整改建议。

## 依赖与降级
以既有规则/规范为准；规则不明时先索取依据，不臆断放行高风险项。

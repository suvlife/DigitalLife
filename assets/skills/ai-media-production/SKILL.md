# AI 图像/视频创作

## 适用范围
- 用生成式模型产出图像/视频素材，控制风格与一致性。

## 工作流程
1. 明确用途、画幅、风格基调与品牌约束。
2. 写结构化提示词：主体 + 场景 + 风格 + 光影 + 镜头 + 负向词。
3. 控制一致性：固定种子/参考图/角色设定，分批迭代。
4. 依据分镜或文案组织素材，命名规范、按镜头归档。
5. 筛选与质检：分辨率、瑕疵、版权与合规风险。

## 产物与规范
- 交付素材文件 + 提示词与参数记录，便于复现。
- 产物写入团队工作目录 outputs/，给出相对路径供下载。

## 依赖与降级
依赖可用的图像/视频生成能力；不可用时输出提示词与分镜方案作为降级交付。

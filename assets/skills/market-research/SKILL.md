# 市场与行业研究

## 适用范围
- 做市场规模、竞品、行业趋势、进入策略等研究判断。

## 工作流程
1. 界定市场边界与研究问题，明确 TAM/SAM/SOM 口径。
2. 检索并交叉验证行业数据、政策、头部玩家动态（先经 research-brief）。
3. 用框架分析：SWOT、波特五力、价值链、竞品矩阵。
4. 从需求侧与供给侧判断机会、壁垒与风险。
5. 输出结论 + 依据 + 不确定性 + 建议。

## 产物与规范
- 交付研究报告：市场概览 → 竞争格局 → 趋势 → 机会/风险 → 建议。
- 数据表交 spreadsheet-studio，正式报告交 document-studio。

## 依赖与降级
关键数字必须标来源；缺数据时给出估算方法与假设。

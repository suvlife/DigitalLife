# 数据分析

## 适用范围
- 对结构化数据做清洗、统计、趋势与异常分析。
- 从数据中提炼洞察，给出置信区间与风险提示。

## 工作流程
1. 先核对数据来源、口径、时间范围与缺失/异常情况，不清楚口径不出结论。
2. 明确分析目标后再选方法：描述统计、分组对比、相关性、时间序列、漏斗等。
3. 计算指标时写清公式与样本量，避免以偏概全。
4. 结论必须有数据支撑，标注显著性/置信区间与潜在偏差（采样、幸存者偏差等）。
5. 给出可视化建议（图型选择、维度、对比基准）与下一步分析方向。

## 产物与规范
- 输出「结论 → 支撑数据 → 方法与口径 → 风险与不确定性 → 建议」。
- 表格/图表数据交由 spreadsheet-studio 生成可下载产物。

## 依赖与降级
可结合 Python（pandas/numpy）分析；无运行环境时给出手工可核对的计算步骤与公式。

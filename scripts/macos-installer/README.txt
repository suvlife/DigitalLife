DigitalLife macOS 社区安装包

安装方法：
1. 解压 ZIP。
2. 双击“安装数字人生.command”。
3. 安装器会校验应用完整性、移除下载隔离属性，并安装到 /Applications/DigitalLife.app。

说明：
- 当前社区包使用 Apple ad-hoc 完整性签名，不是 Apple Developer ID 公证包。
- 如果 macOS 首次阻止脚本运行，请按住 Control 点击脚本，选择“打开”；或在“系统设置 → 隐私与安全性”中允许。
- 项目配置 Developer ID 证书与公证凭据后，Release 流程会自动生成可直接双击安装的 Apple 公证包。
